# CSV validation

## Run Code
```
python main.py
```
## Code format

— files:
- The file will contain all of our files

— models: 
- The models package will contain all of our models 

— services: 
- The services package will contain all the business logic of our application 

```
.
├── files 
│   ├── csv.csv  
│   └── validation_rules.json
├── models
│   └── person_model.py          
│   
├── sevices
│   └── person_service.py          
│   
└── main.py
```
